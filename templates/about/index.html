Welcome to Playlist Machine, a tool that lets you generate Spotify playlists based on ("inspired by") a specified artist and song.
You can also choose a genre and set up audio features.

This app is created with Python (Flask), using the Spotify Web API (mostly Recommendations endpoint).

A source code is available on GitHub.
